# T.N.T (Tsague & Ndouma Transmission)

A professional web-based telecom transmission network planning platform. Designed for precision RF planning in African infrastructure contexts.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **View the Project**:
   - Landing Page: `http://localhost:3000`
   - Dashboard: `http://localhost:3000/dashboard`

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Charts**: Recharts
- **Components**: shadcn/ui inspired modular architecture

## 📂 Folder Architecture
- `app/`: Application routes and layouts.
- `components/landing/`: High-conversion landing page sections.
- `components/dashboard/`: Professional engineering dashboard views.
- `components/ui/`: Shared UI components (Buttons, Cards, etc.).
- `lib/`: Utility functions and shared logic.
- `public/`: Static assets and generated imagery.

## ✨ Core Features
- **Interactive Map**: Visualize microwave links across terrain (Leaflet integration ready).
- **Link Budget Calculator**: Precision simulation of path loss and rain fade.
- **Interference Analysis**: AI-driven spectral coordination.
- **Site Management**: Full inventory of towers and equipment.
- **Responsive Design**: Optimized for desktop and field use on tablets.
